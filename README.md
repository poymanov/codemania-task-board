# Учебный проект Task Board

Менеджер задач, похожий по функционалу на Trello - доски, колонки, задачи.

## Использование

Перед запуском необходимо создать `.env` с тестовыми значениями:

```
make create-env
```

После этого можно запустить сервисы:

```
make up
```

Для остановки использовать:

```
make down
```

Для перезапуска:

```
make restart
```

Для просмотра логов **docker-compose**:

```
make logs
```
